# Prashanth Nemadi - Portfolio

A modern, animated portfolio website showcasing my work as a Software AI Engineer and Full-Stack Developer.

## 🌐 Live Demo
[View Portfolio](https://prashanthnemadi18.github.io/portfolio)

## 🚀 Features
- Modern, bright design with smooth animations
- Responsive layout for all devices
- Separate detailed About page
- Project showcase with images
- Skills, Education, and Achievements sections
- Contact information

## 🛠️ Technologies Used
- HTML5
- CSS3 (with animations)
- JavaScript
- Gradient backgrounds
- Modern UI/UX design

## 📦 Deployment

### Deploy to GitHub Pages:
1. Create a new repository on GitHub named `portfolio`
2. Push these files to the repository
3. Go to Settings > Pages
4. Select "main" branch as source
5. Your site will be live at: `https://prashanthnemadi18.github.io/portfolio`
