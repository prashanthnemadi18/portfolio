Honeypot-Based Cyberattack Detection System

Built a honeypot-driven system to capture, analyze, and classify cyberattacks using machine learning and real-time dashboards.

  • Built a data collection pipeline in Python to parse honeypot logs and store structured data in MySQL for further analysis.
  • Designed a feature extraction and training pipeline using Scikit-Learn to train a Random Forest–based detection model.
  • Developed a Flask API to serve model predictions and built a React + Tailwind dashboard to visualize live alerts and explainable AI (XAI).

Tech: Python, Flask, Scikit-Learn, MySQL, React, Tailwind CSS, Docker, Cowrie
