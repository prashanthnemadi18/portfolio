## 🔧 To See the Images

### Step 1: Restart the Development Server
```bash
cd portfolio-react
npm start
```

### Step 2: Clear Browser Cache
- Press `Ctrl + Shift + R` (Windows/Linux)
- Or `Cmd + Shift + R` (Mac)
- Or open DevTools (F12) → Right-click refresh button → "Empty Cache and Hard Reload"

### Step 3: View the Project
1. Navigate to your portfolio at `localhost:3000`
2. Scroll to the Projects section
3. Click "View more" on the Smart Hire ATS project
4. You should see all 14 images in the gallery
5. Click any image to zoom in with the image name displayed below
